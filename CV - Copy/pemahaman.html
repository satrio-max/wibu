Pemahaman Pengenalan Pemograman :

  • (0) Tidak Tahu
  • (1) Tahu tapi belum mengerti
  • (2) Mengerti tetapi belum dipraktikkan
  • (3) Sudah dipraktikkan

    • HTML

    NO MATERI PENGENALAN PEMOGRAMAN (HTML) LEVEL PEMAHAMAN MATERI
    1. pengenalan code HTML
    ( h1,h2,h3,p,bold,i, dan lain-lain )     		(3) Sudah dipraktikkan
    2. HTML tabel
    ( tHead,tBody,tFoot,tr,td )     		(3) sudah dipraktikkan
    3. Hyperlink & Image
    (a=href & img=src)     		(3) sudah dipraktikkan
    4. Menampilkan data
    (Header,Menu,Nav,Main,Article,Section,Display inline block,div dan span)     		(3) sudah dipraktikkan

    PHP

    NO MATERI PENGENALAN PEMOGRAMAN (PHP) LEVEL PEMAHAMAN MATERI
    1. PHP
    ( Pengenalan XAMPP dan cara Penggunaannya )     		(3) Sudah dipraktikkan
    2. Variabel (2) mengerti tapi belum dipraktikkan
    3. Type Data
    		 (2) mengerti tapi belum dipraktikkan
    4. Operator
    (Aritmatika,Assignment,Increment & Decrement,Relasi, Logika,Bitwise,Ternary)     		(1) tahu tapi belum mengerti
    5. If/else
    		 (2) mengerti tapi belum dipraktikkan
